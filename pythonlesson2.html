Умови та цикли


Умови (if, else) - це спосіб приймати рішення у програмі, виконуючи код лише тоді, коли виконується певна умова.
Цикли (for, while) дозволяють повторювати блоки коду кілька разів, поки виконується певна умова.

1. Умова if

a = 10

if a > 5:
    print("a більше за 5")

2. if – else

age = 16

if age >= 18:
    print("Ти повнолітній")
else:
    print("Ти ще неповнолітній")

3. if – elif – else

mark = 9

if mark >= 10:
    print("Відмінно")
elif mark >= 7:
    print("Добре")
else:
    print("Потрібно вчитися краще")

4. Логічні оператори

a = 5
b = 10

if a > 0 and b > 0:
    print("Обидва числа додатні")

if a < 0 or b < 0:
    print("Хоча б одне число від'ємне")

if not a == b:
    print("Числа не рівні")

5. Цикл for

for i in range(5):
    print(i)

6. Цикл while

i = 1
while i <= 5:
    print(i)
    i += 1

7. break та continue

for i in range(1, 10):
    if i == 5:
        break
    print(i)

for i in range(1, 10):
    if i == 5:
        continue
    print(i)
