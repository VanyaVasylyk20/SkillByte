HTML

Що таке HTML


HTML (HyperText Markup Language) — мова розмітки,
яка створює структуру веб-сторінок.

HTML:
- не мова програмування
- описує вміст сторінки
- працює з CSS і JavaScript

Структура HTML-документа


<!DOCTYPE html>
<html>
  <head>
    <title>Назва сторінки</title>
  </head>
  <body>
    Вміст сторінки
  </body>
</html>

HTML-теги


Тег записується в кутових дужках.

Приклади:
<p>Абзац</p>
<h1>Заголовок</h1>
<a>Посилання</a>

Більшість тегів мають
відкривальний і закривальний тег

Текстові теги


Заголовки:
<h1> — найбільший
<h6> — найменший

Текст:
<p>Абзац</p>
<b>Жирний</b>
<i>Курсив</i>
<br> — перенос рядка

Списки


Маркований список:
<ul>
  <li>Пункт</li>
</ul>

Нумерований список:
<ol>
  <li>Пункт</li>
</ol>

Посилання та зображення


Посилання:
<a href="https://example.com">Перейти</a>

Зображення:
<img src="image.jpg" alt="Опис">

alt — обов’язковий атрибут

Таблиці


<table>
  <tr>
    <th>Назва</th>
    <th>Оцінка</th>
  </tr>
  <tr>
    <td>Математика</td>
    <td>10</td>
  </tr>
</table>

Форми


<form>
  <input type="text" placeholder="Ім'я">
  <input type="password">
  <button>Надіслати</button>
</form>
