Математика

1. Арифметика

Арифметика — основа всіх обчислень у програмуванні.

  • Додавання: a + b
  • Віднімання: a - b
  • Множення: a * b
  • Ділення: a / b
  • Остача від ділення: a % b

Якщо a % b = 0 — число ділиться без остачі.

2. Алгебра

Алгебра працює зі змінними та формулами.

  • Змінна — контейнер для числа.
  • Формула — математичний вираз зі змінними.

Приклад формули периметра: P = 2 * (a + b).

3. Логіка

Логіка визначає істинність умов.

  • AND — істина, якщо всі умови істинні.
  • OR — істина, якщо хоча б одна умова істинна.
  • NOT — змінює істину на хибу.

4. Булева алгебра

Дозволяє спрощувати складні логічні вирази.

  • !(A && B) = !A || !B
  • !(A || B) = !A && !B

5. Теорія множин

Множина — набір унікальних елементів.

  • Об’єднання — всі елементи з двох множин.
  • Перетин — спільні елементи.
  • Різниця — елементи першої множини без елементів другої.

6. Комбінаторика

Вивчає кількість можливих варіантів.

  • Перестановки — порядок важливий.
  • Комбінації — порядок не важливий.

7. Теорія ймовірностей

Ймовірність події обчислюється формулою:

P = сприятливі випадки / всі можливі випадки.

8. Статистика

Аналізує набори чисел.

  • Середнє значення = сума / кількість.
  • Мінімум та максимум у масиві.

9. Дискретна математика

Працює з об’єктами, які можна порахувати.

  • Множини
  • Графи
  • Таблиці істинності

10. Теорія графів

Граф складається з вершин та ребер.

Використовується для пошуку маршрутів.

11. Математична логіка

Вивчає істинні та хибні твердження.

12. Теорія алгоритмів

Показує, як швидко працює програма.

  • O(1) — миттєво
  • O(n) — залежить від розміру даних
  • O(n²) — повільно

13. Матрична алгебра

Матриця — це таблиця чисел.

Вектор — рядок або стовпець матриці.

14. Геометрія

Точка задається координатами (x, y).

Відстань між точками: √((x₂-x₁)² + (y₂-y₁)²).

15. Бітова математика

Комп'ютер працює з бітами 0 та 1.

  • AND
  • OR
  • XOR
  • Зсуви бітів
