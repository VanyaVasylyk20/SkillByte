Основи мови Java

Що таке Java


Java — це мова програмування, яка працює на будь-якому пристрої завдяки JVM.

JDK — набір інструментів для створення програм.
JRE — середовище для запуску програм.
JVM — віртуальна машина Java.

Структура програми

    public static void main(String[] args) {
        // код програми
    }
}
