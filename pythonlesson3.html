Колекції даних

Колекції даних - це способи зберігати і групувати багато значень разом в одній змінній.Вони потрібні, щоб не створювати десятки окремих змінних для схожих даних.

1. Списки (list)

numbers = [1, 2, 3, 4, 5]
names = ["Іван", "Оля", "Марія"]

print(numbers[0])   # перший елемент
numbers.append(6)   # додати елемент
numbers.remove(3)   # видалити 3
print(len(numbers)) # кількість елементів

2. Кортежі (tuple)

coords = (10, 20)
print(coords[0])
# coords[0] = 5  # помилка, кортеж незмінний

3. Множини (set)

nums = {1, 2, 3, 3, 4}
print(nums)  # повтори автоматично видаляються

nums.add(5)
nums.remove(2)

4. Словники (dict)

student = {
    "name": "Іван",
    "age": 18,
    "grade": 10
}

print(student["name"])
student["age"] = 19
student["city"] = "Одеса"

5. Перебір колекцій

for n in numbers:
    print(n)

for key, value in student.items():
    print(key, "=", value)
