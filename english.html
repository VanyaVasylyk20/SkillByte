Англійська

1. Parts of Speech — Частини мови

  • Noun — bug, code
  • Verb — run, fix
  • Adjective — fast, slow
  • Adverb — quickly
  • Pronoun — it, they
  • Preposition — in, on
  • Article — a, the
  • Conjunction — and, but

2. Articles — a / an / the

a/an — будь-який об’єкт, the — конкретний

I wrote a script. The script works.

3. Plural Nouns — Множина

  • file → files
  • class → classes
  • library → libraries

4. Pronouns — Займенники

I, you, he, she, it, we, they, this, that, these, those

5. Present Simple

Формула: I work / He works

I test code every day.

6. Present Continuous

am/is/are + verb-ing

I am fixing a bug.

7. Past Simple

verb + ed / irregular

I finished the task.

8. Future Tense

will — рішення зараз, going to — план

I will deploy tomorrow.

9. Modal Verbs

can, must, should, may

You should update the project.

10. Conditionals

If + Present → Future

If the app crashes, I will fix it.

11. Imperative

Run the program. Save the file.

12. Questions & Negatives

Do you use Python?

I don’t understand this error.

13. Passive Voice

The project was deployed.

14. Prepositions

in a folder, on Monday, at 6 PM

15. Comparatives

easy → easier → the easiest

Java is harder than Python.

16. Linking Words

and, but, because, so, however

17. Infinitive / Gerund

to code, coding

I like coding.
