Об’єктно-орієнтоване програмування (ООП)

ООП (Об’єктно-орієнтоване програмування) - це підхід до програмування, де програма будується з об’єктів, які мають властивості (дані) та методи (дії), із застосуванням інкапсуляції, наслідування, поліморфізму та абстракції для зручності, зрозумілості та повторного використання коду.

1. Створення класу

class Student:
    pass

2. Конструктор __init__

class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

3. Створення об'єкта

s1 = Student("Іван", 18)
print(s1.name)
print(s1.age)

4. Методи класу

class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def info(self):
        print(self.name, self.age)

5. Наслідування

class CollegeStudent(Student):
    def __init__(self, name, age, college):
        super().__init__(name, age)
        self.college = college

6. Інкапсуляція

class Person:
    def __init__(self, name):
        self.__name = name  # приватне поле

    def get_name(self):
        return self.__name

7. Поліморфізм

class Dog:
    def sound(self):
        print("Гав")

class Cat:
    def sound(self):
        print("Мяу")

animals = [Dog(), Cat()]
for a in animals:
    a.sound()
