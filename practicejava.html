Задачі: Основи мови Java


1. Вивести "Привіт, Java!"
System.out.println("Привіт, Java!");

2. Змінна age і виведення
int age = 18;
System.out.println(age);

3. Сума двох змінних
int a = 5;
int b = 10;
System.out.println(a + b);

Задачі: Умови та цикли


4. Перевірка числа
int x = 12;
if (x > 10) {
    System.out.println("Більше 10");
} else {
    System.out.println("10 або менше");
}

5. Цикл for від 1 до 5
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

6. Вивести парні числа від 2 до 10
int i = 2;
while (i <= 10) {
    System.out.println(i);
    i += 2;
}

8. Масив та перебір елементів

Задачі: Об’єктно-орієнтоване програмування (ООП)


9. Клас Car та об'єкт
class Car {
    String model;
    int year;
}
Car myCar = new Car();
myCar.model = "BMW";
myCar.year = 2020;
System.out.println(myCar.model + " " + myCar.year);
