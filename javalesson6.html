Обробка помилок та робота з файлами


Робота з файлами — це збереження даних на комп’ютері або читання даних з файлу.Файли потрібні, щоб програма могла зберігати інформацію навіть після закриття.
Обробка помилок — це спосіб передбачити можливі проблеми під час виконання програми і правильно на них реагувати, щоб програма не зупинялася раптово.

Обробка помилок: try / catch


try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Помилка: " + e.getMessage());
}

finally


try {
    int[] numbers = {1, 2, 3};
    System.out.println(numbers[5]);
} catch (ArrayIndexOutOfBoundsException e) {
    System.out.println("Помилка індексу");
} finally {
    System.out.println("Цей блок виконається завжди");
}

Створення власного винятку


class MyException extends Exception {
    MyException(String message) {
        super(message);
    }
}

throw new MyException("Щось пішло не так");

Читання файлів: Scanner


File file = new File("data.txt");
Scanner sc = new Scanner(file);
while(sc.hasNextLine()) {
    String line = sc.nextLine();
    System.out.println(line);
}
sc.close();

Читання файлів: BufferedReader


BufferedReader reader = new BufferedReader(new FileReader("data.txt"));
String line;
while((line = reader.readLine()) != null) {
    System.out.println(line);
}
reader.close();

Запис у файл: PrintWriter


PrintWriter writer = new PrintWriter("output.txt");
writer.println("Привіт Java");
writer.close();
