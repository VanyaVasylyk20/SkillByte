Робота з файлами та помилки


Робота з файлами — це збереження даних на комп’ютері або читання даних з файлу.Файли потрібні, щоб програма могла зберігати інформацію навіть після закриття.
Обробка помилок — це спосіб передбачити можливі проблеми під час виконання програми і правильно на них реагувати, щоб програма не зупинялася раптово.

1. Відкриття файлу

file = open("data.txt", "r")
text = file.read()
print(text)
file.close()

2. Запис у файл

file = open("data.txt", "w")
file.write("Привіт, файл!")
file.close()

3. Конструкція with open

with open("data.txt", "r") as f:
    print(f.read())

4. Обробка помилок try / except

try:
    a = int(input("Введи число: "))
    print(10 / a)
except ValueError:
    print("Потрібно ввести число!")
except ZeroDivisionError:
    print("На нуль ділити не можна!")
finally:
    print("Кінець програми")

5. Перевірка існування файлу

import os

if os.path.exists("data.txt"):
    print("Файл існує")
else:
    print("Файл не знайдено")
