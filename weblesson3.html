CSS

Що таке CSS


CSS (Cascading Style Sheets) — це мова стилів,
яка відповідає за зовнішній вигляд веб-сторінки.

CSS:
- задає кольори, шрифти, розміри
- працює разом з HTML
- робить сайт красивим

Підключення CSS


Зовнішній файл (рекомендовано):

<link rel="stylesheet" href="style.css">

CSS-файл:
body {
  background-color: white;
}

CSS-синтаксис


Синтаксис CSS:

селектор {
  властивість: значення;
}

Приклад:
p {
  color: red;
  font-size: 16px;
}

Селектори CSS


Селектор за тегом:
p { color: blue; }

За класом:
.text { color: green; }

За id:
#title { font-size: 24px; }

HTML:
<p class="text">Текст</p>

Кольори та фон


Колір тексту:
color: red;
color: #ff0000;

Фон:
background-color: yellow;
background-image: url("img.jpg");

Текст і шрифти


font-size: 18px;
font-family: Arial;
font-weight: bold;
text-align: center;
text-decoration: underline;

Розміри та відступи


width: 200px;
height: 100px;

Відступи:
margin: 10px;
padding: 15px;

Рамка:
border: 1px solid black;

Блочні та рядкові елементи


display: block;
display: inline;
display: inline-block;
display: none;

Приклади блочних:
<div> <p> <h1>

Рядкові:
<span> <a>

Flexbox (основи)


.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Flexbox використовується
для зручного розміщення елементів

Псевдокласи


a:hover {
  color: red;
}

button:active {
  background-color: gray;
}

Псевдокласи реагують
на дії користувача

Коментарі та поради


Коментар у CSS:
/* Це коментар */

Поради:
- не використовуй inline-стилі
- давай зрозумілі назви класам
- пиши акуратний код
