Об’єктно-орієнтоване програмування (ООП)

ООП (Об’єктно-орієнтоване програмування) - це підхід до програмування, де програма будується з об’єктів, які мають властивості (дані) та методи (дії), із застосуванням інкапсуляції, наслідування, поліморфізму та абстракції для зручності, зрозумілості та повторного використання коду.

Клас та об’єкт


class Car {
    String model;
    int year;
}
Car myCar = new Car();
myCar.model = "BMW";
myCar.year = 2020;

Конструктор


class Car {
    String model;
    int year;

    Car(String m, int y) {
        model = m;
        year = y;
    }
}
Car myCar = new Car("Audi", 2022);

Інкапсуляція


class Person {
    private String name;

    public void setName(String n) {
        name = n;
    }

    public String getName() {
        return name;
    }
}

Наслідування


class Animal {
    void speak() {
        System.out.println("Тварина говорить");
    }
}

class Dog extends Animal {
}

Поліморфізм


Animal a = new Dog();
a.speak();

Абстрактний клас


abstract class Shape {
    abstract double area();
}

interface Flyable {
