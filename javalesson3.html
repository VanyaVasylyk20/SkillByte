Методи та масиви


Масив - це структура даних, яка зберігає багато значень одного типу у впорядкованому порядку під одним ім’ям.
Метод - це блок коду, який виконує певне завдання і може бути викликаний з інших частин програми.

Створення методів


public static void sayHello() {
    System.out.println("Привіт Java");
}

Метод з параметрами


public static void showSum(int a, int b) {
    System.out.println(a + b);
}

Метод з return


public static int sum(int a, int b) {
    return a + b;
}

Перевантаження методів


public static int sum(int a, int b) {
    return a + b;
}

public static double sum(double a, double b) {
    return a + b;
}

Масиви


int[] numbers = {1, 2, 3, 4, 5};

Доступ до елементів масиву


System.out.println(numbers[0]);

Перебір масиву


for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

Багатовимірні масиви


int[][] matrix = {
    {1, 2},
    {3, 4}
};

System.out.println(matrix[0][1]);
