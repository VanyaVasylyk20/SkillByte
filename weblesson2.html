JavaScript

Що таке JavaScript


JavaScript — це мова програмування, яка використовується
для створення інтерактивних веб-сторінок.

JavaScript:
- працює у браузері
- додає логіку та поведінку
- реагує на дії користувача

Змінні та типи даних


Змінні:
let age = 15;
const name = "Іван";

Типи даних:
Number     — числа
String     — текст
Boolean    — true / false
null       — порожнє значення
undefined  — не визначено

Масив:
let numbers = [1, 2, 3];

Об'єкт:
let user = {
  name: "Оля",
  age: 16
};

Оператори, умови та цикли


Оператори:
+  -  *  / 
=== !== > < >= <=
&& || !

Умови:
if (age >= 18) {
  console.log("Дорослий");
} else {
  console.log("Неповнолітній");
}

Цикл for:
for (let i = 0; i < 5; i++) {
  console.log(i);
}

Функції


Звичайна функція:
function sum(a, b) {
  return a + b;
}

Стрілкова функція:
const sumArrow = (a, b) => a + b;

Функції дозволяють повторно
використовувати код.

DOM та події


DOM — це доступ до HTML через JavaScript.

Отримання елемента:
document.getElementById("title");

Зміна тексту:
element.textContent = "Новий текст";

Події:
button.addEventListener("click", () => {
  alert("Кнопку натиснуто");
});

Ввід, вивід та налагодження


Вивід:
alert("Привіт");
console.log("Повідомлення");

Ввід:
let name = prompt("Як тебе звати?");

Коментарі:
// однорядковий
/* багаторядковий */

Для перевірки помилок
використовують консоль (F12).
