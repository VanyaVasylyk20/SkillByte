Інформатика

1. Алгоритми та структури даних

  • Алгоритм – покрокова інструкція для розв’язання задачі.
  • Види: лінійні, розгалужені, циклічні.
  • Структури даних: масив, список, стек, черга, словник, множина.
  • Приклад задач: пошук максимального числа, сортування bubble/quick sort.

2. Основи програмування

  • Змінні і типи даних: int, float, string, boolean.
  • Оператори: арифметичні, логічні, порівняння.
  • Умови: if, elif, else.
  • Цикли: for, while.
  • Функції: блок коду, який можна викликати, приймає аргументи і повертає результат.
    • def sum_numbers(a, b):
    return a + b

3. Об’єктно-орієнтоване програмування (ООП)

  • Клас – шаблон для об’єктів, Об’єкт – екземпляр класу.
  • Атрибути – змінні класу або об’єкта, Методи – функції всередині класу.
  • Інкапсуляція, Наслідування, Поліморфізм.
    • class Animal:
    def speak(self):
        print("Some sound")

class Dog(Animal):
    def speak(self):
        print("Woof!")

4. Робота з файлами

  • Відкриття файлів: "r" – читання, "w" – запис.
  • Формати: txt, csv, json.
  • Закриття файлу: file.close() або with open(...) as file:
  • Обробка помилок: try / except.
    • with open("data.txt", "r") as f:
    content = f.read()

5. Основи операційних систем

  • Процес – виконання програми.
  • Потік – підпроцес одного процесу.
  • Оперативна памʼять – тимчасове зберігання даних.
  • Файлова система – структура файлів і папок.
  • Командний рядок – запуск програм та адміністрування ОС.

6. Бази даних

  • Таблиці: рядки = записи, стовпці = поля.
  • SQL команди: SELECT, INSERT, UPDATE, DELETE.
  • Первинний ключ (PK), Зовнішній ключ (FK).
    • SELECT name, age FROM users WHERE age > 18;

7. Комп’ютерні мережі

  • IP-адреса – унікальна адреса пристрою.
  • Порт – канал комунікації (80 – HTTP, 443 – HTTPS).
  • DNS – перетворює домен у IP.
  • Протоколи: TCP, UDP, HTTP/HTTPS.

8. Інтернет та веб-технології

  • Клієнт – браузер / додаток.
  • Сервер – приймає запити, повертає відповіді.
  • REST API – обмін даними (GET, POST, PUT, DELETE).
  • Формати даних: JSON, XML.
    • import requests
response = requests.get("https://api.example.com/data")
print(response.json())

9. Системи контролю версій

  • Git – контроль змін коду.
  • Основні команди: git init, git add, git commit, git push, git pull.

10. Алгоритмічна складність

  • O(1) – константний час.
  • O(n) – лінійна складність.
  • O(log n) – логарифмічна (бінарний пошук).
  • O(n^2) – квадратична (сортування вставкою).

11. Основи кібербезпеки

  • Хешування паролів (SHA256, bcrypt).
  • HTTPS – шифрування даних.
  • Захист від SQL ін’єкцій та XSS.
  • Антивірус, оновлення системи.

12. Налагодження та тестування

  • Debugging – пошук та виправлення помилок.
  • Логування – print або логери.
  • Unit-тести – перевірка окремих функцій.
  • Інтеграційні тести – перевірка взаємодії модулів.

13. Архітектура програм

  • MVC: Model – дані, View – інтерфейс, Controller – логіка.
  • Модульність – код поділений на файли та функції.
  • Переваги: зрозумілість, повторне використання, легше тестування.

14. Основи штучного інтелекту

  • Машинне навчання – навчання програми на даних.
  • Класифікація – визначення типу (spam/not spam).
  • Регресія – прогноз чисел (ціна, температура).
  • Приклади: рекомендації, чат-боти, розпізнавання облич.
