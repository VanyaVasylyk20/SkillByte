Функції та модулі


Функція - це фрагмент коду, який виконує певне завдання і може використовуватися багато разів у програмі.
Модуль - це файл з кодом, який містить функції, класи або змінні, які можна імпортувати та використовувати в інших програмах.

1. Створення функцій

def say_hello():
    print("Привіт!")

2. Параметри функцій

def greet(name):
    print("Привіт,", name)

greet("Іван")
greet("Оля")

3. Повернення значень

def add(a, b):
    return a + b

result = add(5, 3)
print(result)

4. Локальні та глобальні змінні

x = 10

def test():
    x = 5
    print(x)

test()
print(x)

5. Імпорт модулів

import math
print(math.sqrt(16))

from random import randint
print(randint(1, 10))

6. Власний модуль

# файл mymodule.py
def hello():
    print("Hello from module!")

7. Використання власного модуля

import mymodule
mymodule.hello()
